Android Calculator

This is a basic calculator built using Android and Java.

MIDI Playback and Modification Program

Using C, I built a program that is capable of parsing through and modifying MIDI files to change octaves, notes, and other significant events. The GUI was built using GTK.

Solar Tracking Weather Station

This project is a solar powered and tracking weather station that reports the weather in the surrounding environment of the device and uploads that data to the Losant cloud platform. It was built using C/C++ on a Photon microcontroller, custom PCB boards, and CAD software.

Currency Converter Java Application

Using Kotlin, I built a web currency converter that uses the fixer.io API to provide real time exchange rates for currency conversion.
